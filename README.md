## 简介
jsUtil是通过作者平时开发积累和一些现网上零散整理的常用javascript函数的封装，方便使用。持续更新...<br>
欢迎大家来为jsUtil添砖加瓦，**希望jsUtil能成为一个大家共同维护的共享库**。 [立即发起PR→](https://github.com/cwlch/jsUtil)  [我要留言](https://github.com/cwlch/jsUtil/issues)
